# 🧮 Chess Game

## Description
A chess game that can be played locally from your device. 

## Challenges
- Right now, this chess game has one major issue, and it's that you can accidentally eat your own piece. Try to add a way to prevent this
- It's quite confusing to tell which pieces are white and black. Try adding images to each piece to make the distinction better.
