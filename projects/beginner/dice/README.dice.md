# 🧮 Dice generator

## Description
An application that asks for an input of dice and generates a random number from 1 to 6 for each dice.

## Challenges
- There is currently no error handling, so inputting a negative value or a non-integer value breaks the code. How can we fix this?
- Output the average of all dice rolled via the console
