# 🧮 Tic Tac Toe game

## Description
A TicTacToe game that allows users to play locally with another player, via the terminal

## Challenges
- Find a way to allow the users to keep track of each player's scores
- Allow the use of the option to play on a 3x3 board or a 4x4 board.
